# ME vs Software Careers at IIT Mandi

**Q: Can Mechanical Engineering students get software jobs?**

- Yes, but the transition is harder than from EE or CSE. ME students must self-learn all core programming and DSA.
- Most software companies allow ME students for placements, but some restrict to circuital branches (CSE/EE/DSE).
- ME students who want software roles should start coding early, join coding clubs, and build a strong project portfolio.

**Q: How hard is the transition?**

- Hard. ME curriculum has little programming. You must self-study DSA, algorithms, and system design.
- Success stories exist, but require exceptional self-motivation and preparation.

**Q: What are the main challenges?**

- Competing with CSE/EE/DSE students who have more coursework and coding experience.
- Some companies filter by branch or require coding contest ranks.

**Tips:**
- Start LeetCode/CP early, do software internships, and build a GitHub portfolio.
- Network with seniors who have made the transition.

**Summary:**
- Software transition from ME at IIT Mandi is possible, but requires significant extra effort and is less common than from circuital branches.
