# Glossary — IIT Mandi Counseling

**JoSAA:** Joint Seat Allocation Authority (centralized counseling for IITs/NITs)
**PSU:** Public Sector Undertaking (government company)
**GATE:** Graduate Aptitude Test in Engineering
**DSA:** Data Structures & Algorithms
**OA:** Online Assessment (placement test)
**CGPA:** Cumulative Grade Point Average
**SOP:** Statement of Purpose (for MS/PhD)
**PPO:** Pre-Placement Offer
**ISTP:** Interactive Socio Technical Project
**MTP:** Major Technical Project
**SURGE:** IIT Kanpur research internship program
**MITACS:** Canadian research internship program
**DAAD:** German research internship program
**CP:** Competitive Programming
**Core:** Traditional engineering (EE, ME, CE, EP)
**Circuital:** CSE, DSE, EE (sometimes MnC)
**Non-circuital:** ME, CE, EP, BioE, Chem, Agri, Quantum, IMBA
