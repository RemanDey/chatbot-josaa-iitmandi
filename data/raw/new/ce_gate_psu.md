# CE GATE and PSU Pathways at IIT Mandi

**PSU Optionality:**
- CE students can target PSUs like AAI, NHAI, CPWD, ONGC, NHPC, etc.
- GATE CE rank under 1000 (sometimes under 500) is needed for top PSUs.

**Preparation Tips:**
- Start GATE prep by 3rd year; focus on core CE subjects: Structures, Geotech, Transportation, Environment, etc.
- Use standard books and join a test series.

**Summary:**
- CE is a good branch for PSU jobs via GATE at IIT Mandi, but competition is intense and private core jobs are fewer.
